SQLSyntaxErrorException

Chamado 77052

O incidente acontece quando se realiza uma consulta ao banco de dados para obter informações de uma pessoa. Durante essa consulta, são fornecidas informações em formato de texto que devem ser convertidas em números. Se, por algum motivo, essa conversão não for possível, uma exceção chamada java.sql.SQLSyntaxErrorException é acionada.

É importante mencionar que já existe mecanismos de replace e tratamentos para que o campo consultado que é texto em tela, chegue no objeto DAO apenas com caracteres numéricos, para o incidente não acontecer, então, testando a consulta da tela, não foi possível replicar o incidente, entretanto será feita uma validação apenas para garantir que seja informado o tipo de dados correto durante a pesquisa

Solução de Contorno: Não se aplica.

Solução Definitiva: devera ser feito no método br.gov.mt.seguranca.srop.negocio.PessoaViewBO#consultaIndividuoUnico uma conversão dos dados de consulta para garantir que seja passado apenas tipo numérico
