SSTBM

More than one row with the given identifier was found

Chamado 78249

Foi identificado que ao fazer a busca de um determinado Alvara ao carregar os dados sobre a DeclaracaoIluminacao acontece a exceção HibernateException: More than one row with the given identifier, ela indica que estava esperando apenas um elemento e esta sendo carregado mais de um.

Nesse caso existe dois registros DeclaracaoIluminacao associado ao registro do Alvara de ID = 47275.

Solução de Contorno: Não se aplica.

Solução Definitiva:

  1. Excluir um dos registros eliminando a duplicação

  2. modificar a tabela DSCIP_OWNER.DSCIP_DECLARACAO_ILUMINACAO no banco de dados para que ele aceite somente um único registro id_alvara
